SKY AI Project

SKY is an advanced personal AI assistant inspired by futuristic concepts like J.A.R.V.I.S. and Wakandan tech. Created by Ibrahim Quartermain, SKY is designed to be a modular, voice-powered, intelligent system that works both offline and online across multiple platforms.

Project Modules

  • 🔒 Security & Anti-Theft System
  • 🧠 Memory & Custom Voice Commands
  • 📂 File Transfer & Bluetooth Control
  • 📆 Reminder & Scheduler
  • 📡 Cloud Sync & Offline-Online Switching
  • 🌍 Location Detection & Smart Reporting
  • 🎙️ Voice Activation with "Hey SKY"

Technologies Used

  • MIT App Inventor (Android App)
  • Arduino Mega + PCA9685 + Sensors
  • HC-05 Bluetooth module
  • MPU6050 Gyroscope + Ultrasonic sensors
  • Kiswahili + English voice command support

Mission: To bring intelligent assistance, automation, and learning into everyday Kenyan lives through DIY innovation and powerful local language support.

Development Goals

  1. Finish all SKY Modules (Privacy, Scheduler, Cloud, etc.)
  2. Build the Android app and PC control panel
  3. Integrate AI with smart robots and smart home systems
  4. Launch a GitHub or learning platform for others

If you're interested in contributing, testing, or following development updates, feel free to reach out or follow on social media.
